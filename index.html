Scalable Vector Graphic (SVG)

What are SVGs?

Scalable Vector Graphics (SVG) are resolution-independent images that stay crisp and clear at any size. As two-dimensional graphics defined using XML, SVGs are perfect for modern web design—supporting responsiveness, animation, and interactivity.

You can use SVGs in several ways: reference them with the <img> tag in HTML, embed them as CSS backgrounds using background-image: url('image.svg');, or include them directly in your HTML as inline SVG code.

Inline

The following examples showcase basic types of SVGs that can be created using XML. These shapes are added using the inline method, meaning they are embedded directly within the HTML file. Below is an example of the structure: 

                        
     <html>
         <head>
         </head>
         <body>
             <svg>
                 <circle/>
                 <rect/>
             </svg>
         </body>
     </html>

Some pros for using this are:

  • Style using CSS/Javascript
  • Alter anytime you want
  • Use for light design

Embed SVG Using <img>

While you can create SVG shapes by embedding them directly into HTML, this approach can make the code look messy—especially with more complex shapes, which can be difficult to read and manage. Because of this, it’s often better to embed SVGs using the <img> tag.
